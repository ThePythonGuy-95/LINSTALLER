# LINSTALLER
An utility to install ISOs for various Linux distros easily.

# SUPPORTED DISTROS
Version 1.5 supports the following Linux distros:
- Ubuntu
- Debian
- Linux Mint
- Arch Linux
- OpenSUSE
- Fedora
